# 🔐 Vaultsy Chrome Extension

A lightweight and secure Chrome extension for generating strong passwords directly from your browser.

Built as an extension of the **Vaultsy** project, this tool provides quick access to secure password generation without requiring external services, accounts, or internet connectivity.

---

## ✨ Features

* 🔒 Generate strong, random passwords instantly
* 🔤 Includes uppercase and lowercase characters
* 🔢 Includes numbers and special symbols
* ⚡ Fast and lightweight user experience
* 🌐 Works completely offline
* 🛡️ No data collection or tracking
* 🎯 Simple, focused interface

---

## 🚀 Installation

### Install from Source

1. Clone the repository:

```bash
git clone https://github.com/jbrown-js/vaultsy-extension.git
```

2. Open Chrome and navigate to:

```text
chrome://extensions
```

3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the project folder.
6. Pin the extension to your toolbar for easy access.

---

## 💡 Usage

1. Click the **Vaultsy** extension icon.
2. Press the **Generate** button.
3. A secure password will be created instantly.
4. Copy and use the generated password wherever needed.

## 🔐 About Vaultsy

The Vaultsy Chrome Extension is based on the **Vaultsy** project, bringing its password generation functionality directly into Google Chrome.

The goal of Vaultsy is to make password security simple, accessible, and convenient. This extension provides a streamlined way to generate secure passwords whenever and wherever they're needed.

Your privacy and security are a priority.
